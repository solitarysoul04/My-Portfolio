# Anand Kumar Pankaj - Developer Portfolio

A minimalist, highly optimized personal developer portfolio built to showcase projects in Artificial Intelligence, Embedded Systems, and Robotics. 

## 🚀 Live Demo
https://my-portfolio-omega-three-uehdkpoa0a.vercel.app/

## 🛠️ Tech Stack
* **Structure:** HTML5
* **Styling:** Tailwind CSS + Custom Vanilla CSS
* **Interactivity:** Vanilla JavaScript (ES6+)
* **Assets:** FontAwesome 6.4.0

## ✨ Key Features
* **Zero-Dependency Interactivity:** Custom-built typewriter effect and scroll-reveal animations using the modern `IntersectionObserver` API—no heavy JS animation libraries required.
* **Responsive Architecture:** Fully responsive CSS grid layout that gracefully degrades from desktop to mobile, complete with a custom vanilla JS mobile navigation toggle.
* **Modern Developer Aesthetic:** Clean, flat charcoal (`#1a1a1a`) and amber (`#fbbf24`) UI prioritizing readability, high contrast, and performance over bloated visual effects.
* **Optimized Deployment:** Structurally separated codebase designed to bypass strict Cross-Origin Read Blocking (CORB) and render-blocking issues on cloud hosting platforms.

## 🗂️ Project Structure
The project is kept intentionally lightweight and separated into core web fundamentals:
* `index.html` - Core markup and Tailwind utility classes.
* `style.css` - Base resets, custom scrollbar styling, and smooth card hover transitions.
* `script.js` - DOM manipulation, event listeners, and asynchronous UI logic.

## 🧠 Why Vanilla?
While frameworks like Next.js or React are incredible tools, I wanted to ensure this portfolio was as lightweight and fast as possible. Building the logic and styling from scratch with core web technologies allowed for total control over the DOM, zero bloat, and a much deeper practical understanding of browser rendering and deployment debugging.
